# ChronoGen: Intelligent Timetable Management System

An automated university timetabling solution using an evolutionary genetic algorithm to solve the NP-hard "Course Timetabling Problem."

## 🚀 Quick Start
1. Ensure **XAMPP** and **Python 3** are installed.
2. Open the `saoo` folder.
3. Run **`run.bat`**.
4. Access the app at: **http://localhost:8000/login.php**

## 📂 Project Structure
- **/saoo**: The PHP-based web interface (built on Gentelella Admin).
- **/timetable-generator**: The Python AI engine that generates conflict-free schedules.

## 🛠 Features
- **Genetic Algorithm**: Optimized (1+1) Evolutionary Strategy for schedule generation.
- **Glassmorphism UI**: Modern dashboard with real-time analytics.
- **Demo Mode**: Works without a database for quick testing.
- **Notification System**: Automated teacher alerts for period changes.

## 📝 Requirements
- PHP 7.4+
- Python 3.8+
- Modern Web Browser
